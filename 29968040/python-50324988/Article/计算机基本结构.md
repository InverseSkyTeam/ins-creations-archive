# 计算机的组成
- 计算机：硬件和软件组成
- 硬件: 运算器、控制器、存储器、输入设备、输出设备
- 软件：系统软件、应用软件
- 中央处理器(Center Process Unit)：运算器、控制器、寄存器（主频和字长）

# 存储器

### 内(主)存储器
- 属于主机的一部分，是临时存储器，存放当前程序
- 内存认为 RAM(随机存储器) 、 ROM(只读存储器)、Cache(高速缓冲存储器)
- RAM：是一种读写存储器，断电后数据消失
- ROM：只读出数据，不能写入或更改新的内容，断电后不丢失，用于检查计算机配置，提供输入输出控制程序
- Cache：CPU和内存之间的高速小容量存储器，固化在主板上（满足高速CPU）
- 数据 -> RAM -> Cache -> 寄存器 ->CPU
- 速度： 寄存器 > Cache > RAM > ROM

### 外存储器
- 有软盘、硬盘、闪盘、光盘、U盘等

# 总线
- 用于传输信息
- 数据总线（DB）：双向的，输送信息，链接CPU各个部件
- 地址总线（AB）：传递地址信息，单向的
- 控制总线（CB）：传控信号（中断申请，准备就绪）

# 性能指标
- 字长：能同时处理的二进制代码的位数，常用16、33、64位
- 运算速度：每秒中能执行的指令条数，MIPS为单位（百万每秒）
- 主频：时钟频率，越高越快，单位MHz/DHz
- 内存容量：内存能存储的字节总数，2、4、8GB

# CPU（中央处理器）
- CPU的最大存储量取决于地址总线
- ALU：CPU的组成部分(算术逻辑单元)
- Inter Ltanium、AMDAthon64、AMD Opteron、IBM Power5
